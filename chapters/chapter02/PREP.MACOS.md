## Prerequisites

* [homebrew](https://brew.sh/)

## Installation

This is pretty straight forward. Open a terminal and run the following. Open a terminal:

```
$ brew install haskell-platform haskell-stack
```


Tested on
* [Sierra](https://www.apple.com/macos/sierra/)
